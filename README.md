## Install

```bash
$ npm i egg-socket.io-emitter --save
```

## Usage

```js
// {app_root}/config/plugin.js
exports.socketIoEmitter = {
  enable: true,
  package: 'egg-socket.io-emitter',
};
```

[socket.io-emmiter](https://github.com/socketio/socket.io-emitter) for egg
